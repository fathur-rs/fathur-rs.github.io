GITHUB PAGE TUTORIAL

Step 1: Make a Website

Of course, before that we have to create a website. create a simple static website using Bootstrap. Bootstrap will handle all the backend hasle



Step 2: Make a Github Repository

Create a new Github repository with a special naming format (your github username).github.io



Step 3: Git

Initiate your Github Repo to your local workspace

git init

git add .

git commit -m "message"

Push the code to your Github Page Repo

git remote add origin https://github.com/(your_username)/(your repository)

git push -u origin master



Step 4: Done

Yay!, your Page has successfully hosted on Github Page
